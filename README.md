# 🎬 Movie Guide App

A simple and responsive Movie Guide App built using **HTML, CSS, and JavaScript**. Users can search for any movie and get detailed information instantly.

---

## 🚀 Features

* 🔍 Search for any movie
* 📄 View movie details (Title, Rating, Plot, etc.)
* 📱 Fully responsive design (Mobile + Tablet + Desktop)
* ⚡ Fast and user-friendly interface

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

---

## ⚙️ How It Works

1. User enters a movie name in the search bar
2. App fetches data from a movie API
3. Displays movie details on the screen
